# featuredrop docs

featuredrop is a headless, framework-friendly feature discovery toolkit.

## What you get

- Feature badges and changelog primitives
- Guided onboarding (tours, hotspots, checklists)
- Adapter-based persistence (memory, local storage, remote, database adapters)
- Built-in schema validation and CI-friendly checks

## Recommended path

1. Start with [Quickstart](/docs/quickstart)
2. Copy one flow from [Examples](/docs/examples)
3. Check [API](/docs/api) for lower-level adapters and hooks
