# Examples

Use these as copy-paste starting points.

## Sidebar "What's New" button

- Place `<NewBadge />` beside your nav entry
- Open `<ChangelogWidget />` from that button click

## Feature launch sequence

- Start with a banner announcement
- Follow with a short tour
- Finish with spotlight for a high-value action

## Adapter strategies

- `MemoryAdapter` for demos/tests
- `RemoteAdapter` for central manifest control
- `IndexedDBAdapter` + `HybridAdapter` for offline-first behavior
